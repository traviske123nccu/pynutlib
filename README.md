# pynutlib

A nutrition scoring and visualization tool built with Python + Streamlit, leveraging the USDA FoodData Central API.

## Install

```bash
!pip install pynutlib
```

## Usage

```python
from pynut import search_usda_foods, calculate_tee, score_menu
```
